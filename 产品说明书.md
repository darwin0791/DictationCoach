# 正字 / DictationCoach 产品说明书

> 文档版本：1.6
> 产品形态：macOS 原生桌面应用  
> 适用版本：0.3.0（当前 MVP）
> 更新日期：2026-07-18

## 1. 产品简介

**正字（DictationCoach）**是一款面向个人英语学习者的 macOS 原生听写工具。

它内置人教 PEP 2012 版和 PEP 2024 版三至六年级教材词库，支持英文听写、中文默写、错词复听和教材常用句整理。用户也可以导入自己的单词材料，通过系统语音完成练习，由用户人工判断每个单词是否正确，并自动沉淀错词、安排复听，形成简单清晰的学习闭环。

产品不依赖账号体系，也不需要联网完成核心学习流程。教材数据、常用句、练习记录和听写进度均保存在用户本机。

应用启动时会先播放约 4.6 秒的品牌进场动画，动画结束后自动进入主界面。

## 2. 产品目标

正字聚焦四个目标：

1. **快速开始听写**：减少复杂配置，从已有单词或教材范围直接开始练习。
2. **集中解决错词**：答错的单词自动进入错题集，避免遗漏薄弱项。
3. **形成可持续复习**：错词连续答对 3 次后标记为“基本掌握”，让复习结果清晰可见。
4. **贴合教材复习**：使用已核对的教材词汇和常用表达，按年级、册、单元组织学习内容。

## 3. 适用人群

- 需要日常进行英语单词听写的学生。
- 希望按人教版 PEP 年级、册、单元复习单词的学习者。
- 有自定义词表、课堂词表或截图材料，需要快速导入练习的人。
- 希望集中查找和跟读教材常用句、表达和谚语的学习者。
- 注重本地数据、希望使用轻量个人工具的 macOS 用户。

## 4. 核心学习闭环

```text
添加或导入单词
      ↓
选择听写方式、练习模式与教材范围
      ↓
播放发音并完成听写
      ↓
人工判断“对”或“错”
      ↓
错词自动进入错题集
      ↓
错题复听，连续答对 3 次
      ↓
标记为“基本掌握”
```

## 5. 页面与功能

应用采用左侧固定导航，从上到下依次为：**听写、单词本、常用句、错题集、设置**。

听写、单词本、常用句和错题集进入后先展示教材选择页。“人教版 3–6 年级（PEP 2012）”支持全部四个模块；“人教版 3–6 年级（新）/ PEP 2024”已开放听写、单词本和错题集，因为尚无新版句子数据，其常用句卡片仍显示“准备中”。“人教版 7–9 年级”当前也为“准备中”。选择教材后，功能页会显示当前教材路径和“更换教材”入口。再次点击左侧模块会重新回到教材选择页，不显示“上次使用”批注。

### 5.1 听写

听写页用于选择练习范围、播放发音和记录对错。

#### 听写方式

- **英文听写**：默认方式，播放当前英文单词，学习者根据英文发音完成拼写。
- **中文默写**：只播放当前词条的教材中文释义，学习者根据中文写出英文。
- 两种方式在点击开始或播放时均连续播放 3 次，每次启动间隔 3 秒；切换单词、停止或重新播放会取消尚未完成的旧播放任务。
- 听写开始后，听写方式会锁定；停止当前练习后才可切换。
- 英文和中文标签使用对应语言图标；未开始时，播放按钮下方分别提示“点击播放，开始英文听写”或“点击播放，开始中文默写”。
- 中文默写提供“显示英文提示”开关，并与听写方式切换放在同一行。关闭时隐藏卡片中的英文单词和音标；打开时显示完整答案。
- 中文语音始终只朗读清理后的教材中文释义，不会朗读释义中可能泄露答案的英文词形说明。

#### 听写范围

- **全部单词**：从当前筛选范围内的全部单词中开始听写。
- **错题复听**：只练习错题集中尚未“基本掌握”的单词。
- 可按**要求、年级、册、单元**筛选人教版 PEP 教材词汇。
- 要求筛选包括：全部、会写、认读、未标注。默认仍由用户自行选择练习范围，不自动限定为会写词。
- 开始听写后，练习队列按当前模式和筛选条件生成。

#### 听写操作

- 未开始时，点击练习卡片中的播放按钮开始。
- 开始后可使用“上一个、播放、下一个、停止”。
- 鼠标移动到练习卡片左侧会显示“对”，点击后记录为正确。
- 鼠标移动到练习卡片右侧会显示“错”，点击后记录为错误。
- 判断完成后自动进入下一个单词。
- 听写卡片使用当前教材义项；单词本和错题集中的释义则继续使用 ECDICT 通用释义。

#### 快捷键

| 操作 | 快捷键 |
| --- | --- |
| 上一个 | 左方向键 |
| 下一个 | 右方向键 |
| 播放发音 | 空格 |
| 停止听写 | Esc |

#### 续练

听写进行中会自动保存听写方式、英文提示状态、练习模式、教材筛选、练习队列和单词位置。意外关闭窗口后，再次打开应用可恢复上次进度。

### 5.2 单词本

单词本用于管理全部学习单词，并提供三种导入方式。

#### 手动新增

- 输入单个英文单词后添加。
- 应用会自动查询本地词典并补充音标、中文释义和例句等信息。
- 如果输入不像英文，或本地词典没有收录，应用会提示确认；用户仍可选择继续添加。
- 已存在的重复单词不会再次添加。
- 新增前必须在当前教材内具体选择年级、册和单元；范围不完整时会提示并保留输入内容。

#### 批量导入

- 支持一次粘贴多个单词。
- 可使用换行、空格、逗号、分号或制表符分隔。
- 导入时自动跳过重复项，并反馈成功与跳过数量。
- 如果英文单词已经存在但尚未归属当前年级、册和单元，系统会给原词追加教材归属，不创建重复词条。

#### OCR 图片导入

- 支持选择 PNG、JPG、TIFF、BMP、GIF、HEIC 等图片。
- 使用 macOS Vision 在本地识别图片中的英文单词。
- 识别完成后先展示预览，再由用户确认导入。
- 年级、册或单元未具体选择时不会写入数据，OCR 预览会保留。

#### 单词管理

- 搜索英文、中文释义、音标、词性释义、例句或教材信息。
- 按要求、年级、册、单元筛选教材词汇。
- 播放单词发音。
- 编辑单词并重新匹配词典信息。
- 删除单词及其对应练习记录。
- 产品不提供手动修改音标或中文释义的能力，避免教材释义与词典信息混用。

### 5.3 常用句

常用句页用于整理和跟读教材中的常用句、表达和谚语，布局和主要管理方式与单词本保持一致。

- 基准数据共 285 条，其中**常用句与表达 263 条、谚语 22 条**。
- 支持搜索英文、中文释义和教材信息。
- 支持按年级、册、单元和内容类型筛选。
- 支持播放英文句子。
- 支持手动新增、文本批量导入和 OCR 图片识别导入。
- 支持编辑和删除；用户修改后的句子数据单独保存在本机。

### 5.4 错题集

错题集集中展示听写中答错过的单词。

- 答错后自动加入，无需手动收藏。
- 优先展示仍在复习中的词，并记录错误次数和连续答对次数。
- 支持播放发音。
- 支持与单词本一致的搜索能力。
- 支持按学习要求筛选：全部要求、会写、认读、未标注。
- 支持按掌握状态筛选：全部状态、复习中、基本掌握。
- 支持按年级、册、单元筛选。
- 支持将当前搜索和筛选结果导出为 CSV；导出字段包括单词、音标、中文释义、学习要求、掌握状态、错误次数、连续答对、年级、册和单元，可直接用 Excel 或 Numbers 打开。
- 已“基本掌握”的错词仍保留可见，便于回顾历史学习情况。

### 5.5 设置

设置页用于调整听写语音偏好。

#### 发音人（英文）

- 美式女声
- 美式男声
- 英式女声
- 英式男声

应用会优先选择当前 macOS 中匹配的系统语音；如果首选语音不可用，则回退到相同语言区域的可用语音。

#### 发音人（中文）

- **高级女声 Yue**：默认选项，使用 macOS 的 `Yue (Premium)` 语音包。
- **标准女声 Tingting**：保留为可选的标准普通话女声。

中文语音按 Apple 语音包标识符匹配，避免系统显示名称本地化后选错语音。如果当前电脑未安装 Yue，应用会回退到可用的普通话女声。

#### 语速

- 正常
- 慢速

设置页为英文和中文分别提供试听按钮，可随时预览当前语音效果。

## 6. 错题与掌握规则

### 判断为“错”

- 错误次数加 1。
- 自动加入错题集。
- 掌握状态变为“复习中”。
- 错题连续答对次数归零。
- 更新最近答错时间和最近练习时间。

### 判断为“对”

- 正确次数加 1。
- 如果该词已经在错题集中，连续答对次数加 1。
- 连续答对不足 3 次时，状态保持“复习中”。
- 连续答对达到 3 次时，状态变为“基本掌握”。

### 状态说明

| 状态 | 含义 |
| --- | --- |
| 新词 | 尚未进入错题复习流程 |
| 复习中 | 曾经答错，仍需要继续复听 |
| 基本掌握 | 进入错题集后已连续答对 3 次 |

当前版本不提供单独的“移出错题集”操作。删除单词时，该单词及其学习记录会一并删除。

## 7. 本地词典与教材数据

### 本地词典

应用优先使用内置精简 ECDICT SQLite 词典查询：

- 美式或英式音标。
- 常用中文释义。
- 词性释义。

精简词典由完整 ECDICT 原料库抽取生成，覆盖教材词、教材句子中出现的词、中小学/基础标签词和高频词；教材中 ECDICT 没有收录的短语会使用教材释义生成兜底词条。

如果 SQLite 词典未命中，则使用轻量 JSON 词典兜底。复数、`-ed`、`-ing` 等词形缺少音标时，应用会尝试从基础词形补充音标，同时保留原词自己的中文释义。

### 教材分类

内置人教版 PEP 2012 和 PEP 2024 三至六年级教材词表索引，支持先选择教材版本，再按年级、册、单元定位单词。

当前教材基准对应用户已核对确认的人教 PEP 2012 版三至六年级数据，共 **816 条教材记录、785 个不同单词或短语**。早期公开网页词表只保留为历史来源，不再用于覆盖已核对数据。

PEP 2024 版运行词库由完整合并表生成。原表 847 条记录经英文拼写清理后接入 845 条、836 个不同单词或短语，其中会写 110 条、认读 414 条、未标注 321 条；当前覆盖三年级上册至六年级上册，尚无六年级下册数据。两条原表已注明需核对原图的记录暂不进入运行词库。新版目前没有常用句数据。

应用首次切换到新教材基准时，会先备份原单词数据，再完成词条迁移：

- 教材中相同的词保留正确次数、错误次数、错题状态和最近练习时间。
- 新教材基准缺少的历史普通词从单词本归档。
- 新教材基准缺少的历史错词仍保留在错题集，不会因迁移被删除。

听写页面只展示当前年级、册、单元中的教材释义，不使用普通词典释义兜底；单词本和错题集继续展示 ECDICT 通用释义。产品不提供手动修改音标或中文释义的能力。

教材短语会进行组件匹配。例如 `ice cream` 对应的教材信息也可关联到 `ice` 和 `cream`。未匹配教材索引的自定义单词仍可正常使用，但启用教材筛选后不会显示。

## 8. 数据存储与隐私

核心功能均在本机完成：

- 单词和练习数据保存在：

  ```text
  ~/Library/Application Support/AIEnglishDictationCoach/words.json
  ```

- 听写会话进度保存在：

  ```text
  ~/Library/Application Support/AIEnglishDictationCoach/practice_session.json
  ```

- 常用句及用户修改保存在：

  ```text
  ~/Library/Application Support/AIEnglishDictationCoach/sentences.json
  ```

- 教材基准首次迁移前，原单词数据会备份为：

  ```text
  ~/Library/Application Support/AIEnglishDictationCoach/words_before_pep2012_migration.json
  ```

- 词典查询使用应用内置数据库。
- OCR 使用 macOS Vision 本地识别。
- 发音使用 macOS 系统语音能力。
- 当前版本不要求注册、登录或同步云端账号。

如需备份学习数据，可备份上述应用数据目录。删除该目录会丢失已有单词和练习记录。

## 9. 视觉与交互风格

正字采用克制的拟物纸张风格：

- 暖黄渐变纸张背景和轻纸卡。
- 深墨色正文、蓝色墨水提示和红笔批注感。
- 左侧固定浅色导航，保持功能入口稳定。
- 回形针、铅笔和叠纸作为低存在感装饰，不干扰操作。
- 全局文字使用本地 `ukai.ttc` 字体，系统图标保留 SF Symbols 字体。

整体体验强调安静、专注和低配置负担，不采用管理后台或复杂学习平台式界面。

## 10. 系统与运行要求

- 操作系统：macOS 13 或更高版本。
- 应用形态：原生 SwiftUI 桌面应用。
- 推荐启动方式：

  ```bash
  open build/DictationCoach.app
  ```

- 重新构建并打包：

  ```bash
  ./scripts/build_app.sh
  open build/DictationCoach.app
  ```

应用运行时只打包精简 ECDICT 词典，不再内置完整 800MB 级原始词典，因此安装包体积主要由字体、图标和少量本地数据构成。

## 11. 当前 MVP 边界

当前版本专注于教材单词听写、中文默写、错词复听和常用句管理，不包含以下能力：

- 用户账号、会员或云端同步。
- 移动端“拍错题”工作流。
- AI 语法诊断、作文批改或知识地图。
- 自动识别用户手写答案并判断正误。
- 单独移出错题集。
- 多设备学习进度同步。

这些能力不属于当前 MVP，除非后续产品范围明确扩展。

## 12. 推荐使用流程

首次使用可按以下顺序完成一次完整学习：

1. 打开“单词本”，手动添加、批量粘贴或通过 OCR 导入单词。
2. 检查单词的音标、释义和教材标签。
3. 如需复习教材表达，打开“常用句”，按教材范围查找、播放或导入句子。
4. 打开“设置”，分别试听英文和中文发音人，选择语速。
5. 打开“听写”，选择“英文听写”或“中文默写”。
6. 选择“全部单词”或“错题复听”，再按需要选择年级、册和单元。
7. 点击播放开始。中文默写时可通过“显示英文提示”控制答案是否出现在卡片中。
8. 听到发音后在纸上完成拼写，再点击卡片左侧“对”或右侧“错”。
9. 完成一轮后进入“错题集”，查看错误次数和掌握状态。
10. 继续使用“错题复听”，直到错词连续答对 3 次并标记为“基本掌握”。

## 13. 常见问题

### 点击播放后没有声音

- 先确认 Mac 未静音，且当前输出设备可正常播放声音。
- 在“设置”中切换其他发音人并试听。
- 部分系统语音可能尚未安装，应用会尝试回退到同语言区域的可用语音。

### 中文语音仍显示或播放为 Tingting

- 在“设置 → 发音人（中文）”中选择“高级女声”，并点击右侧试听按钮。
- 高级女声需要 macOS 已安装 `Yue (Premium)` 语音包；未安装时应用会回退到 Tingting 等可用普通话声音。
- 可在 macOS“辅助功能 → 朗读内容 → 系统声音 → 管理声音”中下载高级普通话语音。

### 中文默写为什么看不到英文单词

这是“显示英文提示”关闭时的正常行为，用于避免提前看到答案。打开该开关即可显示英文单词、音标和完整教材释义。无论开关是否打开，中文语音都不会朗读释义中的英文词形说明。

### 听写范围显示为 0 个

- 检查单词本中是否已有单词。
- 点击“重置”清除年级、册、单元筛选。
- 使用“错题复听”时，只有尚未标记为“基本掌握”的错词会进入练习范围。

### OCR 没有识别出单词

- 优先使用文字清晰、方向正常、对比度较高的图片或截图。
- 当前 OCR 只提取英文单词，不会把整段排版原样导入。
- 识别结果会先显示预览，确认内容后再点击导入。

常用句页的 OCR 会按识别文本行生成句子预览，更适合教材表达或句子截图。

### 应用体积为什么不再包含完整 ECDICT

当前版本只内置约数 MB 的精简 ECDICT 运行库。完整 ECDICT 仅作为开发期原料，用于重新抽取词典或做英文拼写粗筛，不随应用打包。

### 如何备份或迁移学习记录

退出应用后，备份整个以下目录：

```text
~/Library/Application Support/AIEnglishDictationCoach/
```

恢复时将备份内容放回同一路径。覆盖前建议先保留当前目录副本，避免误删已有记录。
