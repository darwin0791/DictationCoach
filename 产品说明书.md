# 正字 / DictationCoach 产品说明书

> 文档版本：1.0  
> 产品形态：macOS 原生桌面应用  
> 适用版本：0.1.0（当前 MVP）  
> 更新日期：2026-06-20

## 1. 产品简介

**正字（DictationCoach）**是一款面向个人英语学习者的 macOS 单词听写工具。

它帮助用户把自己的单词材料导入本地，通过系统英文语音完成听写练习，由用户判断每个单词是否听写正确，并自动沉淀错词、安排复听，形成简单清晰的学习闭环。

产品不依赖账号体系，也不需要联网完成核心学习流程。单词、练习记录和听写进度均保存在用户本机。

## 2. 产品目标

正字聚焦三个目标：

1. **快速开始听写**：减少复杂配置，从已有单词或教材范围直接开始练习。
2. **集中解决错词**：答错的单词自动进入错题集，避免遗漏薄弱项。
3. **形成可持续复习**：错词连续答对 3 次后标记为“基本掌握”，让复习结果清晰可见。

## 3. 适用人群

- 需要日常进行英语单词听写的学生。
- 希望按人教版 PEP 年级、册、单元复习单词的学习者。
- 有自定义词表、课堂词表或截图材料，需要快速导入练习的人。
- 注重本地数据、希望使用轻量个人工具的 macOS 用户。

## 4. 核心学习闭环

```text
添加或导入单词
      ↓
选择听写模式与教材范围
      ↓
播放发音并完成听写
      ↓
人工判断“对”或“错”
      ↓
错词自动进入错题集
      ↓
错题复听，连续答对 3 次
      ↓
标记为“基本掌握”
```

## 5. 页面与功能

应用采用左侧固定导航，从上到下依次为：**听写、单词本、错题集、设置**。

### 5.1 听写

听写页用于选择练习范围、播放发音和记录对错。

#### 听写范围

- **全部单词**：从当前筛选范围内的全部单词中开始听写。
- **错题复听**：只练习错题集中尚未“基本掌握”的单词。
- 可按**年级、册、单元**筛选人教版 PEP 教材词汇。
- 开始听写后，练习队列按当前模式和筛选条件生成。

#### 听写操作

- 未开始时，点击练习卡片中的播放按钮开始。
- 开始后可使用“上一个、播放、下一个、停止”。
- 鼠标移动到练习卡片左侧会显示“对”，点击后记录为正确。
- 鼠标移动到练习卡片右侧会显示“错”，点击后记录为错误。
- 判断完成后自动进入下一个单词。

#### 快捷键

| 操作 | 快捷键 |
| --- | --- |
| 上一个 | 左方向键 |
| 下一个 | 右方向键 |
| 播放发音 | 空格 |
| 停止听写 | Esc |

#### 续练

听写进行中会自动保存当前模式、教材筛选、练习队列和单词位置。意外关闭窗口后，再次打开应用可恢复上次进度。

### 5.2 单词本

单词本用于管理全部学习单词，并提供三种导入方式。

#### 手动新增

- 输入单个英文单词后添加。
- 应用会自动查询本地词典并补充音标、中文释义和例句等信息。
- 如果输入不像英文，或本地词典没有收录，应用会提示确认；用户仍可选择继续添加。
- 已存在的重复单词不会再次添加。

#### 批量导入

- 支持一次粘贴多个单词。
- 可使用换行、空格、逗号、分号或制表符分隔。
- 导入时自动跳过重复项，并反馈成功与跳过数量。

#### OCR 图片导入

- 支持选择 PNG、JPG、TIFF、BMP、GIF、HEIC 等图片。
- 使用 macOS Vision 在本地识别图片中的英文单词。
- 识别完成后先展示预览，再由用户确认导入。

#### 单词管理

- 搜索英文、中文释义、音标、词性释义、例句或教材信息。
- 按年级、册、单元筛选教材词汇。
- 播放单词发音。
- 编辑单词并重新匹配词典信息。
- 删除单词及其对应练习记录。

### 5.3 错题集

错题集集中展示听写中答错过的单词。

- 答错后自动加入，无需手动收藏。
- 优先展示仍在复习中的词，并记录错误次数和连续答对次数。
- 支持播放发音。
- 支持与单词本一致的搜索能力。
- 支持按年级、册、单元筛选。
- 已“基本掌握”的错词仍保留可见，便于回顾历史学习情况。

### 5.4 设置

设置页用于调整听写语音偏好。

#### 发音人

- 美式女声
- 美式男声
- 英式女声
- 英式男声

应用会优先选择当前 macOS 中匹配的系统语音；如果首选语音不可用，则回退到相同语言区域的可用语音。

#### 语速

- 正常
- 慢速

设置页提供试听按钮，可随时预览当前语音效果。

## 6. 错题与掌握规则

### 判断为“错”

- 错误次数加 1。
- 自动加入错题集。
- 掌握状态变为“复习中”。
- 错题连续答对次数归零。
- 更新最近答错时间和最近练习时间。

### 判断为“对”

- 正确次数加 1。
- 如果该词已经在错题集中，连续答对次数加 1。
- 连续答对不足 3 次时，状态保持“复习中”。
- 连续答对达到 3 次时，状态变为“基本掌握”。

### 状态说明

| 状态 | 含义 |
| --- | --- |
| 新词 | 尚未进入错题复习流程 |
| 复习中 | 曾经答错，仍需要继续复听 |
| 基本掌握 | 进入错题集后已连续答对 3 次 |

当前版本不提供单独的“移出错题集”操作。删除单词时，该单词及其学习记录会一并删除。

## 7. 本地词典与教材数据

### 本地词典

应用优先使用内置 ECDICT SQLite 词典查询：

- 美式或英式音标。
- 常用中文释义。
- 词性释义。
- 可用的中英文例句。

如果 SQLite 词典未命中，则使用轻量 JSON 词典兜底。复数、`-ed`、`-ing` 等词形缺少音标时，应用会尝试从基础词形补充音标，同时保留原词自己的中文释义。

### 教材分类

内置人教版 PEP 三至六年级教材词表索引，支持按年级、册、单元定位单词。

教材短语会进行组件匹配。例如 `ice cream` 对应的教材信息也可关联到 `ice` 和 `cream`。未匹配教材索引的自定义单词仍可正常使用，但启用教材筛选后不会显示。

## 8. 数据存储与隐私

核心功能均在本机完成：

- 单词和练习数据保存在：

  ```text
  ~/Library/Application Support/AIEnglishDictationCoach/words.json
  ```

- 听写会话进度保存在：

  ```text
  ~/Library/Application Support/AIEnglishDictationCoach/practice_session.json
  ```

- 词典查询使用应用内置数据库。
- OCR 使用 macOS Vision 本地识别。
- 发音使用 macOS 系统语音能力。
- 当前版本不要求注册、登录或同步云端账号。

如需备份学习数据，可备份上述应用数据目录。删除该目录会丢失已有单词和练习记录。

## 9. 视觉与交互风格

正字采用克制的拟物纸张风格：

- 暖黄渐变纸张背景和轻纸卡。
- 深墨色正文、蓝色墨水提示和红笔批注感。
- 左侧固定浅色导航，保持功能入口稳定。
- 回形针、铅笔和叠纸作为低存在感装饰，不干扰操作。
- 全局文字使用本地 `ukai.ttc` 字体，系统图标保留 SF Symbols 字体。

整体体验强调安静、专注和低配置负担，不采用管理后台或复杂学习平台式界面。

## 10. 系统与运行要求

- 操作系统：macOS 13 或更高版本。
- 应用形态：原生 SwiftUI 桌面应用。
- 推荐启动方式：

  ```bash
  open build/DictationCoach.app
  ```

- 重新构建并打包：

  ```bash
  ./scripts/build_app.sh
  open build/DictationCoach.app
  ```

内置 ECDICT 数据库体积较大，打包后的应用会包含词典副本，因此应用体积明显大于一般轻量工具。

## 11. 当前 MVP 边界

当前版本专注于英语单词听写，不包含以下能力：

- 用户账号、会员或云端同步。
- 移动端“拍错题”工作流。
- AI 语法诊断、作文批改或知识地图。
- 自动识别用户手写答案并判断正误。
- 单独移出错题集。
- 多设备学习进度同步。

这些能力不属于当前 MVP，除非后续产品范围明确扩展。

## 12. 推荐使用流程

首次使用可按以下顺序完成一次完整学习：

1. 打开“单词本”，手动添加、批量粘贴或通过 OCR 导入单词。
2. 检查单词的音标、释义和教材标签。
3. 打开“设置”，选择发音人和语速，并试听效果。
4. 打开“听写”，选择“全部单词”或“错题复听”。
5. 按需要选择年级、册和单元，点击播放开始。
6. 听到发音后在纸上或心中完成拼写，再点击卡片左侧“对”或右侧“错”。
7. 完成一轮后进入“错题集”，查看错误次数和掌握状态。
8. 继续使用“错题复听”，直到错词连续答对 3 次并标记为“基本掌握”。

## 13. 常见问题

### 点击播放后没有声音

- 先确认 Mac 未静音，且当前输出设备可正常播放声音。
- 在“设置”中切换其他发音人并试听。
- 部分系统语音可能尚未安装，应用会尝试回退到同语言区域的可用语音。

### 听写范围显示为 0 个

- 检查单词本中是否已有单词。
- 点击“重置”清除年级、册、单元筛选。
- 使用“错题复听”时，只有尚未标记为“基本掌握”的错词会进入练习范围。

### OCR 没有识别出单词

- 优先使用文字清晰、方向正常、对比度较高的图片或截图。
- 当前 OCR 只提取英文单词，不会把整段排版原样导入。
- 识别结果会先显示预览，确认内容后再点击导入。

### 应用体积为什么较大

应用内置约 812MB 的离线 ECDICT 数据库，打包时会复制进应用，因此这是当前离线查词方案的正常结果。

### 如何备份或迁移学习记录

退出应用后，备份整个以下目录：

```text
~/Library/Application Support/AIEnglishDictationCoach/
```

恢复时将备份内容放回同一路径。覆盖前建议先保留当前目录副本，避免误删已有记录。
