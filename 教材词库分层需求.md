# 教材词库分层需求

## 1. 一句话判断

应当在现有“要求 / 年级 / 册 / 单元”之前增加“教材词库”层，让家长先选孩子实际使用的教材，再在该教材内选择年级、册和单元。这是扩展新版小学和初中词汇前必须建立的基础数据结构。

## 2. 需求复述

- 产品想法：单词本不再默认展示所有教材词汇，而是先选择一套“教材词库”。
- 目标用户：使用人教版英语教材的小学和初中学生家长。
- 用户目标：快速找到孩子当前教材、年级和单元对应的单词，避免不同版本混用。
- 产品目标：建立可扩展的多教材词库架构，为后续加入新版小学、初中以及句子数据做准备。
- 核心场景：家长进入单词本，选定教材词库，再按年级、册、单元查看或发起听写。

## 3. 名称与词库范围

一级字段统一命名为“教材词库”，不命名为“单词类型”。它表示一套有明确出版社、学段和版本的教材数据。

首期规划三套词库：

| 界面名称 | 内部标识示例 | 学段 | 状态 |
|---|---|---|---|
| 人教版 3–6 年级 | `pep-primary-2012` | 小学 | 已有数据，对应已核验的 PEP 2012 版 816 条记录 |
| 人教版 3–6 年级（新） | `pep-primary-2024` | 小学 | 单词已接入：845 条记录、836 个不同词/短语；常用句待准备 |
| 人教版 7–9 年级 | 待根据教材年份确定 | 初中 | 待导入和人工核验 |

### 命名原则

- 界面沿用用户确认的“（新）”名称，卡片同时显示稳定版本 `PEP 2024`，内部 ID 固定为 `pep-primary-2024`。
- 内部标识一旦发布不随界面名称变化，用于保存筛选、练习进度和数据迁移。
- 每套词库必须记录出版社、教材版本、学段、覆盖年级、数据状态和词条数量。

## 4. 信息结构

页面层级调整为：

```text
左侧主导航
├─ 听写 → 教材选择页 → 该教材的听写页
├─ 单词本 → 教材选择页 → 该教材的单词列表与导入页
├─ 常用句 → 教材选择页 → 该教材的句子列表与导入页
├─ 错题集 → 教材选择页 → 该教材的错词列表
└─ 设置 → 现有设置页
```

不再将“听写列表 / 全部单词 / 全部常用句 / 全部错词”作为四个主模块的默认首页。用户必须先进入教材选择页，明确本次要操作的教材词库。

进入某套教材后，页内筛选层级为：

`要求 → 年级 → 册 → 单元`

其中：

- “教材词库”在上一级选择页中必选、单选，不再和年级、册、单元并列为一个普通下拉筛选。
- “要求 / 年级 / 册 / 单元”为该词库内的可选筛选。
- 返回教材选择页并切换教材后，要求、年级、册和单元自动重置为“全部”，防止残留上一词库的无效选项。
- 年级、册、单元的候选项只来自当前词库，不展示其他词库的值。
- 四个教材选择页都不提供“全部教材词库”，避免同一英文词在不同教材中的义项和学习要求发生冲突。

### 4.1 教材选择页

四个模块复用同一套教材卡片和布局，但保留各自的页头和进入动作。

选择页包含：

- 当前模块页头，例如“听写”或“单词本”。
- 一句明确引导：“选择孩子当前使用的教材”。
- 教材卡片区，默认展示“人教版 3–6 年级”、“人教版 3–6 年级（新）”和“人教版 7–9 年级”。
- 每张卡片显示教材名称、学段、版本状态和与当前模块相关的数量。
- 已可用卡片整卡可点击；进入动作根据当前模块显示为“开始听写”、“查看单词”、“查看常用句”或“查看错题”。
- 未准备完成的卡片显示“准备中”，不响应进入操作。
- 页面保持现有纸张风格，不改成文件夹管理器、表格或后台系统样式。

数量口径按模块区分：

| 模块 | 卡片上的数量 |
|---|---|
| 听写 | 可练习单词数 |
| 单词本 | 该教材词库的单词数 |
| 常用句 | 该教材词库的常用句和谚语数 |
| 错题集 | 该教材词库中已答错过的单词数 |

### 4.2 进入、返回与状态

- 点击左侧的“听写 / 单词本 / 常用句 / 错题集”时，默认进入该模块的教材选择页，不直接打开上次的列表。
- 从教材选择页进入功能页后，页头下方显示“教材词库 / 当前教材名”路径和“更换教材”入口。
- “更换教材”返回当前模块的教材选择页，不切换左侧主导航。
- 教材卡片不显示“上次使用”批注；每次从主导航进入都展示选择页。
- 听写存在未完成会话时，对应教材卡片显示“继续上次听写”，由用户主动继续；不在进入“听写”导航时直接跳过选择页。
- 功能页内的搜索、筛选和编辑状态可在返回教材选择页时清空；已保存的单词、句子和听写会话不受影响。

## 5. 单词本交互

### 5.1 词库选择

- 单词本默认页为教材选择页，不再直接展示“全部单词”列表。
- 选择“人教版 3–6 年级”后，才进入现有的左侧单词列表与右侧导入工具布局。
- 进入功能页后，不再显示“教材词库”下拉框；使用页头下方的当前教材标识和“更换教材”返回上一层。
- “人教版 3–6 年级”与“人教版 3–6 年级（新）”均可在单词相关模块进入；新版常用句因没有句子数据仍为“准备中”。

### 5.2 内容状态

- 已有且已核验的词库可正常选择。
- 已列入规划但数据未导入的词库不伪造空单词列表；如果需要提前展示，则标记“准备中”并禁止选择。
- 词库文件损坏或加载失败时，显示“词库加载失败”，不自动回退到另一套教材。

### 5.3 单词展示

- 列表顶部的数量表示当前教材词库和当前筛选下的词条数。
- 教材标签默认仍显示“年级 + 册 + 单元”；当页面可能混合多个词库时，必须额外显示教材词库名。
- 搜索只搜索当前教材词库，不跨词库返回结果。
- 本次升级时，当前系统内已有的全部单词（包括已有的手动新增、文本导入和 OCR 导入词）默认归入“人教版 3–6 年级”词库，保证升级后不会出现原有单词无处可查的情况。

### 5.4 新增与导入归属

手动新增、批量导入和 OCR 导入统一使用单词本左侧当前选中的教材范围作为导入目标：

```text
当前教材词库 + 当前年级 + 当前册 + 当前单元
```

交互规则：

- 年级、册和单元必须全部选择具体值，任何一项仍为“全部”时都不允许导入。
- 用户点击“新增”、“导入单词”或“确认 OCR 导入”时再做校验，不因未选范围而禁止用户先输入文本或选择图片。
- 如有未选项，不执行写入，并按实际缺失项提示，例如“请先选择年级、册和单元后再导入”或“请先选择单元后再新增”。
- 校验失败时保留用户已输入的文本和 OCR 预览结果，用户补选范围后可直接继续导入。
- 导入成功后，每个单词都新增一条对应当前词库、年级、册和单元的教材归属标签。
- 如果“要求”已选中具体的“会写”、“认读”或“未标注”，则将该值一并写入；如果仍为“全部要求”，则按“未标注”写入，不额外阻断导入。
- 在 OCR 预览到最终确认期间如果用户切换了教材范围，以点击最终导入时的当前选择为准，并在确认区显示目标位置。

重复单词规则：

- 英文单词已存在，但尚未归属当前词库 / 年级 / 册 / 单元时，不创建重复单词，只为原单词追加新的教材归属标签。
- 英文单词和当前教材归属均已存在时，跳过该词并在导入结果中计入“已存在”，不重复写入。
- 批量导入和 OCR 导入完成后显示结果摘要：新增单词数、追加归属数、已存在数和未通过规格检查数。

释义规则：

- 将单词归入某个教材范围，不等于已经获得了可靠的教材释义。
- 手动新增、批量导入和 OCR 导入可继续使用 ECDICT 补充通用释义和音标，但不将 ECDICT 释义冒充为教材释义。
- 没有教材释义的导入词在中文默写中继续显示“未收录教材释义”，不回退播报 ECDICT 释义。

## 6. 与其他页面的联动

本需求先在单词本建立词库维度，但数据模型必须能支持下一步同步到：

- 听写：先选教材词库，再选要求、年级、册和单元；听写会话需保存词库 ID。
- 中文默写：必须播放当前词库、当前单元的教材释义，不能跨版本取义。
- 错题集：可按教材词库过滤；同一英文单词的练习统计首期继续共享，教材归属和释义按当前词库展示。
- 常用句：当前系统内已有的全部常用句（包括内置句子、谚语和用户已导入的句子）在升级时默认归入“人教版 3–6 年级”；未来加入新版句子数据时使用同一套词库 ID，不另建不兼容的版本命名。

## 7. 数据模型要求

每条教材标签在现有字段之外，增加稳定的 `catalogID`：

```text
catalogID + grade + book + unit + meaning + requirement
```

词库元数据至少包含：

```text
id, displayName, publisher, edition, stage, grades, status, vocabularyCount
```

关键规则：

- 不同词库可以包含同一英文单词，但各自保留教材释义、要求、年级和单元。
- 英文单词的通用词典信息仍由 ECDICT 提供，不复制到每套教材词库。
- 教材归属以 `catalogID` 为第一区分条件，不再只靠“三年级 / 上册 / Unit 1”区分。
- 新词库导入不得删除、归档或覆盖旧词库的用户练习数据。
- 现有无 `catalogID` 的全部单词和常用句数据，在升级时统一补为 `pep-primary-2012`；这包括 816 条教材记录、285 条内置常用句基准，以及用户本机已有的自定义单词和句子。
- 本次默认归入是为了保持历史数据可见性，不改写原有单词、句子、教材标签或练习统计。

## 8. MVP 范围

### Must-have

- 建立教材词库元数据和稳定 ID。
- 为现有 PEP 2012 版 3–6 年级数据补充 `catalogID`。
- 将当前系统内全部已有单词和常用句默认迁移至“人教版 3–6 年级”词库。
- 单词本增加“教材词库”选择，并实现年级、册、单元联动。
- 听写、单词本、常用句和错题集的默认页统一改为教材选择页，不直接进入任何列表或练习页。
- 四个功能页都提供当前教材标识和“更换教材”返回入口。
- 手动新增、批量导入和 OCR 导入按当前选中的词库、年级、册和单元写入教材归属，范围不完整时阻止导入并保留已输入内容。
- 默认选中当前已有词库，保证老用户升级后内容和练习记录不丢失。
- 新版和 7–9 年级词库只在数据导入、人工校验并通过完整性检查后启用。

### Should-have

- 记住最后选择的教材词库。
- 在词库选择项中显示学段、版本和词条数。
- 同步在听写和错题集中增加词库筛选，保证三个页面的教材口径一致。
- 对每套词库输出记录数、去重词数、未标注学习要求数和拼写待复核数。

### Not-now

- 不在多套教材之间自动判断孩子应该使用哪一套。
- 不跨词库自动合并教材释义和学习要求。
- 不仅根据 OCR 结果就对新词库标记“已核验”。
- 不在本需求中实现教材在线下载、账号同步或商店化词库。

## 9. 验收标准

1. 老用户升级后，系统内原有的全部单词和常用句都能在“人教版 3–6 年级”中查看，不得丢失、归档或隐藏。
2. 单词的正确次数、错误次数、错题状态和最近练习时间不变；常用句的原文、释义、类型和教材标签不变。
3. 切换教材词库后，年级、册、单元和要求筛选自动重置，候选项只属于新词库。
4. 同一英文单词同时存在于两套词库时，切换词库能分别展示各自的教材释义、年级、单元和学习要求。
5. 未准备好的词库不可进入空列表或发起听写，必须明确显示“准备中”。
6. 启用任一教材筛选后，未归属当前词库的单词不出现；但升级前已存在的用户自定义单词已按迁移规则归入默认词库，应正常显示。
7. 中文默写只能使用当前词库和当前单元的中文教材义，查不到时显示“未收录教材释义”，不从其他词库或 ECDICT 回退。
8. 新词库导入后，原词库记录数、分类标签和用户练习数据不发生意外变化。
9. 年级、册或单元任何一项未具体选择时，三种新增或导入方式均不写入数据，界面准确提示缺失项，且用户已输入的文本或 OCR 预览不丢失。
10. 选定具体词库、年级、册和单元后，三种方式新增的单词都能在对应范围中立即查到。
11. 重复导入同一单词时，系统按“追加教材归属”或“已存在”处理，不创建重复单词，也不清空原练习记录。
12. 点击听写、单词本、常用句或错题集的左侧导航后，首先看到当前模块的教材选择页，不直接看到单词、句子或错题列表。
13. 选择已可用教材后才进入功能页；点击“更换教材”能回到当前模块的选择页，左侧导航选中状态不变。
14. 教材卡片不显示“上次使用”批注，再次点击主导航时回到教材选择页。
15. 听写有未完成会话时，对应教材卡片显示继续入口，由用户点击后才恢复会话。

## 10. 风险与取舍

- **版本命名风险**：“新”不是稳定版本名，正式导入前必须根据教材版权页确认年份或版次。
- **数据混用风险**：只在 UI 加下拉框、不在教材标签加 `catalogID`，会使同年级同单元的不同版本无法区分。
- **迁移风险**：现有逻辑会把教材基准词迁入单词本，如直接合并三套词库，可能导致用户单词本被数千个词条填满；因此必须改为按词库加载和展示。
- **校验成本**：新版小学和初中词汇仍需要走“OCR/抽取 → 英文拼写粗筛 → 人工核对 → 学习要求标注”流程，不能因为有了分层就降低数据标准。

## 11. 产品建议

建议做，但分两步：

1. 已完成教材词库数据模型、选择页、老数据迁移和筛选联动。
2. 已接入新版 3–6 年级单词；下一步补齐新版六年级下册、常用句和 7–9 年级数据，并分别校验后启用。

这样可以先把容器做对，再放入新数据，同时避免未核验词表影响当前可用产品。

## 12. 待确认项

- “人教版 7–9 年级”具体覆盖的教材版本和册数。
- PEP 2024 六年级下册、常用句原始资料，以及 321 条未标注词的人工分类依据。
- 常用句的手动新增和 OCR 导入是否也完全沿用单词的“当前词库 + 年级 + 册 + 单元”归属规则。
